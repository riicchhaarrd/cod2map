cod2map WebAssembly demo

This page runs cod2map locally in the browser. Files remain on this device. Chrome maps a selected Call of Duty 2 folder into a lazy virtual filesystem without copying the whole folder. Firefox uses OPFS as a fallback cache for a selected map/assets subset.

1. Load game files

No files loaded.

The lazy VFS indexes file names and passes Blob handles to the Wasm worker. File contents are loaded only if cod2map opens them. OPFS staging should contain the selected map plus required assets, not necessarily the full game install.

2. Select map

The list shows source maps that can be compiled. map_source/... entries are run with -loadFrom and an output path under raw/maps/..., because cod2map requires a maps path component.
